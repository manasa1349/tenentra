# Frontend (Tenantra)

This is the React + Vite frontend for Tenantra.

## Tech Stack

- React
- Vite
- React Router
- Axios
- React Hot Toast
- Canvas Confetti
- React Spinners

## Key UI Modules

- Public home page (`/`) with long-form product sections
- Authentication pages (`/login`, `/register`)
- Dashboard, projects, tasks
- Tenant/user management views (role-aware)
- Profile and settings pages

## Development

Run from `frontend/`:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Lint

```bash
npm run lint
```
